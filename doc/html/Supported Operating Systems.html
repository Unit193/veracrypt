Supported Operating Systems

VeraCrypt currently supports the following operating systems:
  • Windows 10
  • Windows 8 and 8.1
  • Windows 7
  • Windows Vista
  • Windows XP
  • Windows Server 2012
  • Windows Server 2008 R2 (64-bit)
  • Windows Server 2008
  • Windows Server 2003
  • Mac OS X 10.13 High Sierra
  • Mac OS X 10.12 Sierra
  • Mac OS X 10.11 El Capitan
  • Mac OS X 10.10 Yosemite
  • Mac OS X 10.9 Mavericks
  • Mac OS X 10.8 Mountain Lion
  • Mac OS X 10.7 Lion
  • Linux x86 (32-bit and 64-bit versions, kernel 2.6 or compatible)
  • FreeBSD x86 (32-bit and 64-bit versions, starting from version 11)

Note: The following operating systems (among others) are not supported: Windows RT, Windows 2003 IA-64, Windows 2008 IA-64, Windows XP IA-64, and the Embedded/Tablet versions of Windows.



Also see the section Operating Systems Supported for System Encryption
